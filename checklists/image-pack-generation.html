Image Pack Generation

Category: Creative • SLA: 6h • Version: 0.8

Create on-brand image sets that support articles, briefs, and social posts with consistent styling and metadata.

Inputs Required

  • Content brief or checklist run reference
  • Brand guidelines (colors, typography, logo usage)
  • Image size & format requirements per channel
  • Prompt references for Midjourney/DALL·E
  • Storage destination (Drive folder or CDN)

Steps

1. Define image needs Tool: Airtable brief • SLA: 15 min
Shot list created with use cases, dimensions, style notes, and approval owner.
2. Generate concepts Agent: Midjourney/DALL·E • SLA: 45 min
At least three concept variants per use case exported with prompt + seed logged for reproducibility.
3. Curate & edit Tool: Figma/Photoshop • SLA: 40 min
Final selections color-corrected, cropped, and annotated; layered files saved for future tweaks.
4. Publish image pack Tool: Google Drive / CDN • SLA: 20 min
Images exported to required sizes (web, social, cover); filenames follow convention and stored in Drive folder linked to Artifacts.
5. Document metadata Tool: Airtable Artifacts • SLA: 15 min
Each image logged with alt text, usage rights, aspect ratio, and recommended captions.
6. Close checklist Automation: Airtable update • SLA: 5 min
Run status set to Done, time logged, and pack shared with downstream checklists.

Outputs & Quality Gates

  • Image pack folder with cover + supporting images
  • Prompt log and seeds for regeneration
  • Editable source files (Figma/PSD)
  • Metadata sheet (alt text, captions, usage rights)
  • Checklist retrospective entry for improvements
