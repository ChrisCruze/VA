Research Note → Product Requirements Doc

Category: Research • SLA: 10h • Version: 0.9

Translate exploratory research notes into a validated PRD with user insights, requirements, and delivery milestones.

Inputs Required

  • Research notes (transcripts, highlights, or summary deck)
  • Target persona + problem statement
  • Existing product constraints or design principles
  • Prompt references: MEETING_STRUCTURER, EXEC_SUMMARY
  • Approval routing list (PM, Eng lead, Design)

Steps

1. Normalize research inputs Tool: Airtable Inputs • SLA: 20 min
All source materials attached to Inputs record with tags, personas, and key dates captured.
2. Extract insights Agent: OpenAI • Prompt: MEETING_STRUCTURER • SLA: 25 min
JSON artifact enumerating user goals, pain points, opportunity areas, and supporting quotes saved to Artifacts.
3. Draft PRD outline Agent: OpenAI • SLA: 20 min
Outline includes problem statement, goals, success metrics, requirements, assumptions, and open questions.
4. Author PRD v1 Agent: OpenAI → Google Doc • SLA: 60 min
Draft PRD doc with sections fleshed out, requirement table, risks, and timeline placeholder.
5. Stakeholder review loop Tool: Google Docs comments • SLA: 90 min
Feedback requested from PM, Eng, Design; comments resolved or documented with owners in Run Steps.
6. Version final PRD Tool: Google Docs + Airtable Artifacts • SLA: 15 min
Version number incremented, change log recorded, final link stored with approval flag toggled.
7. Sync tasks & roadmap Tool: Zapier → Airtable/Project tool • SLA: 25 min
Requirements translated into tasks with owners/dates; roadmap updated and linked in Artifacts.
8. Close checklist Automation: Airtable update • SLA: 5 min
Run marked Done, timesheet entries logged, summary shared with leadership.

Outputs & Quality Gates

  • Insight JSON artifact with citations to source notes
  • PRD v1 Google Doc (comment-ready) + final approved version
  • Requirements-to-task mapping sheet
  • Risk register with owners and mitigation status
  • Roadmap link + milestone tracker aligned with PRD
