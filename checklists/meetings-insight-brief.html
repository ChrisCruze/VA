Meeting → Insight Brief

Category: Meetings • SLA: 6h • Version: 1.1

Convert raw meeting conversations into executive-ready briefs with structured insights, action items, and distribution-ready content.

Inputs Required

  • Otter transcript link + exported highlights
  • Meeting metadata (title, date, participants)
  • Associated project / client tag
  • Checklist run owner in Airtable
  • Prompt references for MEETING_STRUCTURER & EXEC_SUMMARY

Steps

1. Ingest transcript Tool: Zapier webhook from Otter • SLA: 5 min
Input row created in Airtable “Inputs” table with transcript, highlights, owner, and status set to New.
2. Structure meeting Agent: OpenAI • Prompt: MEETING_STRUCTURER • SLA: 8 min
JSON payload saved to Artifacts with purpose, agenda, decisions, owners, risks, and follow-ups. Validation check passes.
3. Draft executive brief Agent: OpenAI → Google Doc • Prompt: EXEC_SUMMARY • SLA: 15 min
Google Doc draft (200–300 words) linked to the Run with correct formatting, context, decisions, risks, and next steps.
4. Push action items Tool: Zapier → Airtable Tasks • SLA: 10 min
All owners and due dates logged as individual task records and linked back to the originating Input.
5. Generate distribution summary Agent: OpenAI → Buffer draft • Prompt: SOCIAL_SNIPPETS • SLA: 12 min
Optional LinkedIn-ready summary stored in Distribution table with placeholders for scheduling.
6. QA pass Tools: Hemingway + Grammarly • SLA: 12 min
Readability score ≤ Grade 7 documented; grammar/spell check completed and notes captured in Run.
7. Close the run Automation: Airtable update • SLA: 3 min
Run status updated to Done, total minutes calculated, and artefact links verified.

Outputs & Quality Gates

  • Structured meeting JSON artifact (versioned, checksum optional)
  • Executive brief Google Doc with approval field ready
  • Action items synced to Airtable Tasks with owners/dates
  • Distribution summary draft queued (if requested)
  • QA checklist log with readability score and reviewer notes
