Blog Refinement & SEO

Category: Blogs • SLA: 12h • Version: 1.0

Elevate draft blog content into polished, search-optimized publications supported by visuals and distribution assets.

Inputs Required

  • Source draft (Markdown, Google Doc, or notes)
  • Target topic, audience, and positioning notes
  • SEO_BRIEF and BLOG_DRAFTER prompt references
  • Brand voice guardrails (tone, CTA, product mentions)
  • Distribution channels + publishing credentials

Steps

1. Intake draft into Inputs Tool: Airtable import • SLA: 10 min
Input record created with source link, summary, and owner; status set to Parsed.
2. Generate outline Agent: OpenAI • Prompt: MEETING_STRUCTURER adapted • SLA: 15 min
Outline artifact listing H2/H3 structure with word counts saved to Airtable Artifacts.
3. Build SEO brief Agent: OpenAI • Prompt: SEO_BRIEF • SLA: 20 min
SEO brief JSON stored with keywords, SERP intent, H2/H3 list, competitor URLs, and metadata options.
4. Draft V1 Agent: OpenAI • Prompt: BLOG_DRAFTER • SLA: 40 min
Markdown or Google Doc draft with TL;DR, CTA, and internal link placeholders uploaded to Artifacts.
5. Generate imagery Agent: DALL·E / Midjourney • SLA: 30 min
Cover + 1–3 inline images generated with alt text and stored with file links.
6. Readability & polish Tools: Hemingway, Grammarly • SLA: 25 min
Readability score ≤ Grade 8, grammar issues resolved, voice compliance noted in Run log.
7. Metadata & canonical setup Tool: Airtable + CMS • SLA: 15 min
Canonical URL, meta title/description, OG image, and schema fields populated in CMS draft.
8. Publish & cross-post Tool: CMS + Buffer • SLA: 25 min
Article published on canonical channel, cross-post plan executed, Distribution rows created with scheduled timestamps.
9. Track distribution Tool: Airtable Distribution • SLA: 10 min
Live URLs, UTMs, and initial performance metrics (if available) captured for each channel.
10. Retrospective & time log Tool: Airtable Runs/Timesheets • SLA: 10 min
Run notes document blockers/improvements; time entries submitted and checklist status set to Done.

Outputs & Quality Gates

  • SEO brief artifact with structured JSON and download link
  • Blog draft doc with version + owner metadata
  • Image pack with filenames, alt text, and storage links
  • CMS metadata tracker and publishing confirmation
  • Distribution tracker populated with URLs, UTMs, and schedule
