Prompt Library

Prompt Purpose Primary Output
MEETING_STRUCTURER Extract purpose, agenda, decisions, owners, risks from transcripts. Structured JSON summary
EXEC_SUMMARY Create 200–300 word executive brief with context, decisions, risks, and next steps. Google Doc or Markdown brief
SEO_BRIEF Compile keywords, SERP intent, headings, questions, and competitor references. SEO brief JSON
BLOG_DRAFTER Draft 1,200–1,800 word posts aligned to SEO brief with TL;DR and CTA. Article draft
SOCIAL_SNIPPETS Produce LinkedIn post, X thread, and newsletter blurb tied to canonical URL. Channel-ready copy bundle

Workflow Wiring

  • Triggers: Otter conversations → Inputs, Inputs parsed → Opal job, Artifacts approved → Google Doc templates, Distribution queued → Buffer scheduling.
  • Actions: Run agents with checklist context, persist run steps, upload artifacts, update statuses and timestamps.
  • Error Handling: Failed steps flag runs as Blocked, store error context, and notify stakeholders via Slack or email.

Example Agents

  • Ingestion Agent: Validates Otter link, fetches transcript, and cleans noise.
  • Structurer Agent: Applies MEETING_STRUCTURER to produce JSON insights.
  • Summarizer Agent: Runs EXEC_SUMMARY and publishes to Google Doc.
  • SEO Agent: Builds SEO_BRIEF and passes data to drafting agent.
  • Drafting Agent: Uses BLOG_DRAFTER to produce long-form content with placeholders.
  • Image Agent: Generates visual concepts and alt text.
  • Distribution Agent: Prepares multi-channel copy with UTMs.
  • Analytics Agent: Backfills metrics into Airtable for dashboards.

Security & Governance

  • Store API keys in Opal or a dedicated secrets manager, never inside Airtable fields.
  • Mask personally identifiable information before sending transcripts to third-party models.
  • Version prompts and attach prompt IDs plus versions to each artifact for traceability.
  • Require human approval before distribution to public channels.
