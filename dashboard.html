AI VA Dashboard Blueprint

Visualize checklist throughput, time investment, and publishing outputs in a single interface.

Home System Overview Airtable Model Checklists

Header KPIs

Runs In Progress

Today's active runs vs. due count with SLA countdown per checklist.

Average Minutes / Checklist

Rolling 7-day and 30-day averages derived from Run totals.

Output Velocity

Artifacts produced per week segmented by checklist category.

Publish Rate

Scheduled vs. published ratio with channel breakdown.

Quality Flags

Readability targets, approval status, and QA exceptions aggregated from Runs.

Pipeline Board

Kanban grouping Runs by status (New → In Progress → Review → Done → Blocked) with checklist tags for prioritization.

Time & Load

Timesheet heatmap by person and project, plus totals for today and the current week.

Blend auto-calculated run durations with manual entries for QA or edits.

Run Detail Panel

  • Step timeline with timestamps, owner, duration, and status.
  • Linked artifacts preview (Docs, images, distribution drafts).
  • Current blocker notes and SLA countdown indicator.

Publishing Tracker

Calendar view showing scheduled posts with channel icons, state (queued/published/error), and quick links to URLs.

SEO Performance Panel

List the last 10 posts with primary keywords, SERP rank placeholder, impressions, and click-through metrics.

Search & Filters

Global search by Input, Project, Tag, or Owner. Filters for checklist, status, channel, and SLA range.

Key Formulas

Metric Formula / Source
Total Minutes DATETIME_DIFF(End, Start, 'minutes') aggregated per run.
Checklist Efficiency Total Minutes / Completed Steps
Publish Rate Count(Published) / Count(Scheduled)
Velocity Artifacts per week (Airtable summary block or Looker Studio chart).
Quality Score Weighted average of readability, approvals, and QA checklist completion.
